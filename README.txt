What is it?
===========

lfs_sofortueberweisung is the integration of the payment processor `sofortüberweisung`_ for `LFS`_.

LFS Version
===========

0.7 and beyond.

Documentation
=============

You can find the documentation on `RTD <http://lfs_sofortueberweisung.rtfd.org>`.
